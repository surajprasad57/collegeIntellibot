May I Help You ???

College Info Intellibot project is built using artificial algorithms that analyzes user’s queries and understand user’s message.

This System is a web application which provides answer to the query of the student.

Students just have to query through the bot which is used for chatting.

College Info Intellibot is very useful for students,faculties as well as Parents since

they does not have to go personally to college office for the enquiry.

This web-application enables the students to be updated with college cultural activities.

This web-application saves time for the student as well as teaching and non-teaching staffs.
